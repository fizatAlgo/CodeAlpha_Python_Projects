# CodeAlpha Python Projects 🐍

A collection of beginner-friendly Python projects completed during the **CodeAlpha Internship Program**. These mini-projects demonstrate core Python concepts like loops, conditionals, functions, and data structures.

## 📋 Projects

### 1. **Hangman Game** (`main.py`)
A fun word-guessing game where players have 6 lives to guess a programming language!
- **Features:** Random word selection, input validation, lives tracking
- **Run:** `python main.py`
- **Skills Covered:** `random` module, loops, conditionals, user input

### 2. **Smart Chatbot** (`main (1).py`)
An interactive chatbot that responds to commands and provides real-time information.
- **Features:** Time/date display, greeting based on hour of day, motivational quotes
- **Commands:** hello, time, date, good morning, good night, motivate me, quote, help, bye
- **Run:** `python "main (1).py"`
- **Skills Covered:** `datetime` module, string methods, function definitions, conditional logic

### 3. **Stock Portfolio Tracker** (`main (2).py`)
A simple investment tracker that calculates total portfolio value.
- **Features:** Stock price lookup, investment calculation, portfolio report
- **Run:** `python "main (2).py"`
- **Skills Covered:** Dictionaries, user input, arithmetic operations, control flow

## 🚀 Quick Start

**Requirements:** Python 3.x (no external dependencies needed)

```bash
# Clone the repository
git clone https://github.com/fizatAlgo/CodeAlpha_Python_Projects.git
cd CodeAlpha_Python_Projects

# Run any project
python main.py                    # Hangman
python "main (1).py"              # Chatbot
python "main (2).py"              # Portfolio Tracker
```
